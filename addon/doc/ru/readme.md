## Что нового
Встречайте функцию **Old Log** — интеллектуальный способ мгновенного доступа к логам предыдущих сессий. Больше никакой потери диагностических данных после сбоя или перезагрузки.

## Возможности
**logViewer** превращает стандартный просмотрщик логов NVDA в мощный инструмент для диагностики:

* **Продвинутый поиск:** `Control+F` с поддержкой **регулярных выражений (Regex)** и историей запросов.
* **Мгновенная навигация:** Быстрый переход по результатам через `F3` и `Shift+F3`.
* **Система закладок:** Отмечайте важные события через `Control+F2` и перемещайтесь кнопками `F2` / `Shift+F2`.

## Функция Old Log
* **Прямой доступ:** Извлечение `nvda-old.log` напрямую из системной папки Temp.
* **Горячая клавиша:** `NVDA+Control+L` открывает лог прошлой сессии во внешнем текстовом редакторе.
