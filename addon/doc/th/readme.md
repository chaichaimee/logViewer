# logViewer

**ผู้พัฒนา:** chai chaimee & Pierre-Louis R.  
**Repository:** [https://github.com/chaichaimee/logViewer](https://github.com/chaichaimee/logViewer)

---

## มีอะไรใหม่
พบกับฟังก์ชัน **Old Log** ใหม่—วิธีอัจฉริยะในการกู้คืนและตรวจสอบ Log จากเซสชันก่อนหน้าได้ทันที หมดกังวลเรื่องข้อมูลวิเคราะห์หายหลังการ Restart หรือโปรแกรมค้าง

## คุณสมบัติเด่น
**logViewer** ยกระดับหน้าต่าง Log ของ NVDA ให้เป็นเครื่องมือวิเคราะห์ประสิทธิภาพสูง เพื่อการทำงานที่รวดเร็วและแม่นยำยิ่งขึ้น:

* **ระบบค้นหาขั้นสูง:** กด `Control+F` เพื่อค้นหาด้วย **Regular Expressions (Regex)** พร้อมระบบจำประวัติคำค้นหาล่าสุด
* **นำทางรวดเร็ว:** ใช้ `F3` เพื่อไปข้างหน้า หรือ `Shift+F3` เพื่อย้อนกลับ โดยไม่ต้องเสียเวลาเปิดหน้าต่างใหม่
* **ระบบคั่นหน้า (Bookmark):** ปักหมุดจุดสำคัญด้วย `Control+F2` และเลื่อนไปยังจุดต่างๆ ได้ทันทีด้วย `F2` / `Shift+F2`

## ฟังก์ชัน Old Log
* **เข้าถึงไฟล์ได้ทันที:** ระบบจะดึงไฟล์ `nvda-old.log` จากโฟลเดอร์ Temp มาให้คุณโดยอัตโนมัติ ไม่ต้องหาเองให้เสียเวลา
* **คีย์ลัดอัจฉริยะ:** กด `NVDA+Control+L` เพื่อเปิด Log เซสชันล่าสุดในโปรแกรมแก้ไขข้อความ (External Editor) ที่คุณถนัด

## การรองรับภาษา
**logViewer** รองรับการทำงานร่วมกับ NVDA ทุกภาษาทั่วโลก ด้วยระบบตรวจจับผ่าน **Window Handle (HWND)** ที่มีความแม่นยำสูง มั่นใจได้ว่าใช้งานได้สมบูรณ์ 100% ไม่ว่าคุณจะใช้ภาษาใด
