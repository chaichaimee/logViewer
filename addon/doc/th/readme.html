What's New

ก้าวไปอีกขั้นกับฟังก์ชัน oldLog ระบบเข้าถึงไฟล์บันทึกย้อนหลังที่อัจฉริยะและรวดเร็วกว่าเดิม ช่วยให้คุณไม่พลาดทุกรายละเอียดสำคัญแม้โปรแกรมจะปิดตัวลงอย่างไม่คาดคิด

Features

logViewer ไม่ได้เป็นเพียงแค่เครื่องมือดู Log ทั่วไป แต่คือการอัปเกรดประสิทธิภาพการทำงานของ NVDA Log Viewer ให้ทรงพลังเทียบเท่า Text Editor ชั้นนำ ออกแบบมาเพื่อนักพัฒนาและผู้ใช้ระดับ Power User โดยเฉพาะ:

  • Advanced Search (ระบบค้นหาขั้นสูง):
    เพียงกด Control+F เพื่อสัมผัสประสบการณ์การค้นหาที่เหนือกว่า รองรับทั้ง Case-sensitive, Wrap-around และการใช้ Regular Expressions พร้อมระบบ History จดจำคำค้นหาล่าสุดเพื่อให้คุณทำงานได้ต่อเนื่องไม่มีสะดุด
  • Quick Search Navigation (ทางลัดอัจฉริยะ):
    ไม่ต้องเสียเวลาเปิดหน้าต่างค้นหาซ้ำๆ ใช้ F3 เพื่อไปยังผลลัพธ์ถัดไป หรือ Shift+F3 เพื่อย้อนกลับ สะดวก รวดเร็ว แม่นยำ
  • Bookmark System (ระบบคั่นหน้าอัจฉริยะ):
    ปักหมุดจุดสำคัญใน Log ได้ทันทีด้วย Control+F2 และเคลื่อนที่ไปยังจุดต่างๆ อย่างอิสระผ่าน F2 (ถัดไป) และ Shift+F2 (ก่อนหน้า)

Old Log Functionality

หมดกังวลเรื่องข้อมูลสูญหายเมื่อเครื่องเกิดอาการ Crash ด้วยระบบจัดการ Log ย้อนหลังที่มีประสิทธิภาพสูงสุด

Key Features

  • Instant Access to Previous Session: เข้าถึงข้อมูลจากเซสชันก่อนหน้าได้ทันที โดยระบบจะดึงไฟล์ nvda-old.log จากโฟลเดอร์ Temporary ของระบบมาเปิดให้คุณโดยตรง
  • Seamless Integration: เพียงกดคีย์ลัดเดียว ระบบจะเปิดไฟล์ Log ล่าสุดที่เพิ่งจบการทำงานไปใน Text Editor ตัวโปรดของคุณทันที
  • Precise Debugging: เหมาะสำหรับการวิเคราะห์สาเหตุการทำงานที่ผิดพลาด (Crash Analysis) เนื่องจากไฟล์นี้จะเก็บข้อมูลจนถึงวินาทีสุดท้ายก่อนที่ NVDA จะเริ่มทำงานใหม่
  • Easy Access: เรียกใช้งานได้ง่ายๆ เพียงกด NVDA+Control+L

Handling NVDA Crash and Restart

เมื่อ NVDA เกิดข้อผิดพลาดและต้องเริ่มทำงานใหม่ (Restart) ข้อมูลการทำงานทั้งหมดในเซสชันที่เกิดปัญหาจะถูกจัดเก็บไว้ใน nvda-old.log โดยอัตโนมัติ logViewer จะทำหน้าที่เป็นสะพานเชื่อมให้คุณเข้าถึงไฟล์นั้นได้ทันทีผ่าน External Editor เพื่อให้การตรวจสอบโค้ดและการแก้ปัญหาสามารถทำได้อย่างราบรื่นที่สุด

Language Support

เราออกแบบมาเพื่อผู้ใช้ทั่วโลก logViewer รองรับการทำงานร่วมกับ NVDA ในทุกภาษาอย่างสมบูรณ์

ด้วยเทคโนโลยีการตรวจจับผ่าน Window Handle แทนการอ้างอิงชื่อหน้าต่าง (Title) ทำให้มั่นใจได้ว่าปลั๊กอินจะทำงานได้อย่างแม่นยำ ไม่ว่าคุณจะใช้ NVDA ในภาษาใดก็ตาม
