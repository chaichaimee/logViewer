# logViewer

**Auteurs :** chai chaimee & Pierre-Louis R.  
**URL :** [https://github.com/chaichaimee/logViewer](https://github.com/chaichaimee/logViewer)

---

## Nouveautés
Découvrez la nouvelle fonction **Old Log** : un moyen intelligent de consulter instantanément vos journaux de sessions précédentes. Ne perdez plus jamais vos données de diagnostic après un plantage.

## Fonctionnalités
**logViewer** transforme le Visionneur de journal NVDA en un outil de diagnostic haute performance :

* **Recherche Avancée :** `Control+F` avec support des **Expressions Régulières (Regex)** et historique de recherche.
* **Navigation Rapide :** `F3` pour le suivant, `Shift+F3` pour le précédent, sans rouvrir de dialogue.
* **Marque-pages :** Marquez les points clés avec `Control+F2` et naviguez via `F2` / `Shift+F2`.

## Fonction Old Log
* **Accès Direct :** Récupère le fichier `nvda-old.log` du dossier temporaire système.
* **Raccourci Dédié :** Appuyez sur `NVDA+Control+L` pour ouvrir le log précédent dans votre éditeur de texte externe préféré.
