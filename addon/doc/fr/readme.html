Nouveautés

Découvrez la fonction Old Log : un moyen intelligent de consulter instantanément vos journaux de sessions précédentes.

Fonctionnalités

logViewer transforme le Visionneur de journal NVDA en un outil de diagnostic haute performance :

  • Recherche Avancée : Control+F avec support des Expressions Régulières et historique de recherche.
  • Navigation Rapide : F3 pour le suivant, Shift+F3 pour le précédent, sans rouvrir de dialogue.
  • Marque-pages : Marquez les points clés avec Control+F2 et naviguez via F2 / Shift+F2.

Fonction Old Log

Ne craignez plus de perdre vos logs après un crash.

  • Accès Direct : Récupère le fichier nvda-old.log du dossier temporaire.
  • Raccourci Dédié : Appuyez sur NVDA+Control+L pour ouvrir le log précédent dans votre éditeur de texte externe.
