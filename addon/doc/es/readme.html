Novedades

Presentamos la función Old Log: la forma más inteligente de recuperar y revisar tus registros de sesiones anteriores al instante.

Características

logViewer transforma el visor de registros estándar de NVDA en una herramienta de diagnóstico de alto rendimiento:

  • Búsqueda Avanzada: Usa Control+F con soporte para Expresiones Regulares e historial de búsqueda.
  • Navegación Instantánea: Salta entre resultados con F3 y Shift+F3 sin cerrar diálogos.
  • Marcadores de Precisión: Etiqueta eventos clave con Control+F2 y navega con F2 / Shift+F2.

Función Old Log

Ya no pierdas información tras un cierre inesperado.

  • Acceso Directo: Recupera el archivo nvda-old.log del directorio temporal automáticamente.
  • Activación Rápida: Presiona NVDA+Control+L para abrir el log de la sesión anterior en tu editor externo favorito.

Soporte de Idiomas

Compatible con cualquier idioma de NVDA gracias a la detección mediante Window Handle (HWND).
